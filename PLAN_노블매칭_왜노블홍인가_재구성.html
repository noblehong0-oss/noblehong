"왜 노블홍인가" 풀섹션 재구성 기획

노블매칭 페이지 · 기존 6개 → 신규 10개 강점 · 풀섹션 일관 레이아웃

1. 기존 vs 신규 비교

구분 기존 사이트 (6개) 신규 기획 (10개) 변경
1 커플매니저 1호 CEO 직접 매칭 + 커플매니저 1호 통합분리 기존 1개 → 2개로 분리. CEO 직접 관여 강조 추가
2 공정거래위원회 표준약관 신원인증 및 보안 (약관 포함) 통합 약관 + 보안을 하나로
3 신원 인증 및 보안 (위 항목에 통합) 통합
4 차별화 서비스 3.3.3 프로모션 시스템 유지+강화 막연한 "차별화" → 구체적 시스템명
5 매칭 프로그램 매주 전략회의 + 교제율 데이터 유지+강화 데이터 시각화 추가
6 소비자 선정 명품 브랜드 브랜드만족지수 3년 연속 1위 유지 구체적 수상 실적
7 — 30년 / 4,000쌍 실적 신규
8 — 방송 300회+ 고정 출연 신규
9 — 박사 학위 / 논문 / 저서 신규
10 — 해외거주 자녀결혼 특화 신규
11 — 저출생 캠페인 / 사회공헌 신규
기존 문제점: 6개 강점이 각기 다른 이미지 크기, 폰트, 여백으로 구성 → 일관성 부재, 뒤죽박죽 느낌
개선 방향: 10개 강점을 통일된 풀섹션 레이아웃으로 구성. 좌우 교차 배치 + 일관된 타이포/여백/이미지 비율

2. 풀섹션 일관 레이아웃 규칙

공통 구조 (모든 섹션 동일)

  • 레이아웃: 좌우 5:5 분할 (이미지 50% + 텍스트 50%)
  • 교차 배치: 홀수 섹션 = 이미지 좌 / 텍스트 우, 짝수 섹션 = 텍스트 좌 / 이미지 우
  • 텍스트 영역: 번호 라벨 → 제목 → 서브 카피 → 핵심 수치/뱃지 (있을 경우)
  • 이미지 영역: 16:10 비율, 라운드 코너 12px, 다크 오버레이 가능
  • 섹션 높이: min-height 480px, 패딩 80px 상하
  • 구분선: 섹션 간 골드 그라데이션 얇은 라인
  • 배경: 홀수 = 다크(#1a1714), 짝수 = 크림(#faf9f6) 교차

스크롤 인터랙션

  • 각 섹션 진입 시 이미지 fade-in + 텍스트 slide-up
  • 숫자 카운터가 있는 섹션은 뷰포트 진입 시 카운팅 애니메이션

3. 10개 섹션 상세 기획

STRENGTH 01

직업사전 등록 기준 대한민국 최초 커플매니저. 1996년부터 30년간 오직 한 길.

뱃지: "직업사전 등록 인증" "1996~"
배경: 다크

대형사의 매니저 돌려막기와 차별화. 대표 이름을 건 책임 경영.

핵심 카피: "대표가 직접, 처음부터 끝까지"
배경: 크림

STRENGTH 04

뱃지: "연세대 석사" "박사학위" "논문 2편" "저서 2권"

4. 기존 대비 변경 요약

항목 기존 사이트 신규 기획
강점 수 6개 10개
레이아웃 각기 다른 크기/구성, 일관성 없음 좌우 5:5 교차 풀섹션, 통일된 구조
배경 흰 배경 단일 다크/크림 교차 (프리미엄 느낌)
이미지 크기 제각각, 일부 저화질 16:10 통일, AI 생성 고퀄리티
인터랙션 없음 (정적 페이지) 스크롤 reveal + 카운터 + 호버 효과
데이터 텍스트 서술만 카운터 애니메이션 + 뱃지 시각화
신규 항목 — CEO 직접 매칭, 30년/4000쌍, 방송 300회, 박사연구, 해외특화, 사회공헌
제거 항목 표준약관 (별도), 신원인증 (별도) → 하나로 통합 (STRENGTH 08)

5. 구현 우선순위

  1. Phase 1 — 와이어프레임: 10개 풀섹션 HTML 구조 + 교차 레이아웃 구현
  2. Phase 2 — 이미지: 각 섹션별 AI 이미지 생성 (프롬프트 제공)
  3. Phase 3 — 인터랙션: 스크롤 reveal, 카운터, 호버 효과
  4. Phase 4 — 카피: 최종 문구 검수 및 반영
