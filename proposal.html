01 / Diagnosis

현재 홈페이지 현황 진단

noblehong.com의 기술, 디자인, 콘텐츠, 사용성 전반을 8개 항목으로 분석한 결과입니다. 대부분의 항목에서 2018~2019년 수준에 머물러 있으며, 전면 리모델링이 필요한 상태입니다.

핵심 문제점 상세

모바일 대응 부재

Critical

미디어쿼리 1개(767px)만 존재하며 실제로는 데스크톱 레이아웃이 그대로 축소 표시됩니다. user-scalable=no로 핀치줌까지 차단된 상태입니다. 2026년 기준 모바일 트래픽이 70% 이상인 점을 감안하면 잠재 고객의 절반 이상을 잃고 있습니다.

SEO 구조적 결함

Critical

H1~H6 시맨틱 태그가 단 하나도 없습니다. 35개 이미지 중 alt 태그가 있는 이미지 0개. 검색엔진이 페이지의 주제와 구조를 전혀 파악할 수 없어 검색 노출이 극히 제한적입니다.

노후화된 디자인 트렌드

High

대리석 텍스처 배경, 골드 라인 애니메이션, 다이아몬드(마름모) 카드 UI 등 2018년 경 유행했던 디자인 요소가 그대로 유지되고 있습니다. 영문 제목(ABOUT NOBLE HONG, SPECIAL SERVICE)은 40~60대 타겟 고객층의 가독성을 떨어뜨립니다.

전환 동선 비효율

High

메인 페이지에 상담 신청 폼이 없습니다(form 0개, input 0개). 상담을 하려면 반드시 서브페이지로 이동해야 하며, 이는 전환율을 크게 저하시킵니다. 경쟁사들은 모두 메인 페이지에 인라인 상담 폼을 배치하고 있습니다.

정적 HTML 기술 스택

Major

jQuery + 정적 HTML 구조로 CMS가 없어 콘텐츠 업데이트마다 개발자가 필요합니다. URL 패턴이 /sub01/sub01_01.html 형식으로 SEO에 불리하며, 유지보수 비용이 높습니다.

유튜브 연동 미흡

Major

결혼박사 홍유진TV(구독자 3.03만명, 영상 165개)의 콘텐츠 자산이 홈페이지에 제대로 노출되지 않습니다. 유튜브 클릭 시 새 탭으로 이탈되며, 동영상/쇼츠 구분 없이 단순 링크만 제공됩니다.

벤치마킹 핵심 인사이트

숫자 기반 신뢰 구축

모든 선두업체가 성혼 수, 회원 수, 성공률을 히어로 섹션에 대형으로 표시합니다. 노블홍의 "8,000명 이상 성혼" 실적을 전면에 배치해야 합니다.

메인 페이지 즉시 상담

듀오, 가연, 바로연 모두 메인 페이지 하단 또는 히어로에 간편 상담 폼을 배치합니다. 이름 + 연락처만으로 즉시 상담 신청이 가능한 구조가 표준입니다.

감성 카피 + 전문성

노블레스수현의 "당신이 찾는 사람도 당신처럼 특별해야 합니다" 같은 감성 카피와 홍유진 대표의 30년 전문성을 결합한 스토리텔링이 필요합니다.

03 / Strategy

개선 전략 및 방향성

노블홍의 핵심 자산(30년 경력, 8,000명 성혼, 대표 유튜브 채널)을 극대화하면서 현대적이고 신뢰감 있는 프리미엄 사이트로 전환하는 전략입니다.

포지셔닝 전환

"노블레스 결혼정보회사"에서 "성혼 전문 결혼정보회사"로 메시지 전환. 30년 경력 + 8,000명 성혼 실적을 전면에 배치하여 실적 기반 신뢰를 구축합니다.

노블레스수현, 퍼플스 등 초프리미엄 업체와의 직접 경쟁 대신, "검증된 경력 + 성혼 실적 + 유튜브 전문성"이라는 고유 포지셔닝을 확보합니다.

타겟 명확화

40~60대 싱글/돌싱 타겟에 맞춘 UX 설계. 큰 폰트, 명확한 한글 네비게이션, 불필요한 영문 제목 제거. 전화 상담 CTA를 최우선 배치합니다.

  • 최소 16px 본문 폰트
  • 한글 중심 네비게이션
  • 전화번호 + 카카오톡 항시 노출
  • 3단계 이내 상담 신청 완료

페이지 구조 재설계

기존 6개 대메뉴 + 25개 서브페이지 구조를 3페이지 핵심 구조로 압축합니다.

01

메인 페이지

히어로(성혼 실적) + 대표 소개 + 서비스 개요 + 유튜브 콘텐츠 + 상담 폼 + 미디어 노출

02

콘텐츠 페이지

유튜브 동영상/쇼츠 탭 + 인스타/블로그 피드 + 러브칼럼 + 카테고리 필터

03

상담 신청

위자드 폼(단계별) + 개인정보 동의 + 접수 확인 + 카카오 알림톡 연동

08 / Viral Strategy

바이럴 마케팅 아이디어

홈페이지 유입을 극대화하고 자연 바이럴을 유도하는 인터랙티브 콘텐츠 전략입니다. 듀오의 '내 가입비 테스트'가 바이럴에 성공한 사례를 참고합니다.

MBTI 연애/결혼 매칭 테스트

바이럴 핵심

"나와 매칭 가능성이 높은 MBTI 이성은?" 테스트를 제작하여 SNS 공유를 유도합니다. 결과 페이지에서 노블홍 상담 연결.

  • 10문항 이내 간편 테스트 (3분 소요)
  • 결과: 나의 연애 유형 + 매칭 MBTI + 궁합 점수
  • 카카오톡/인스타 공유 버튼 (바이럴 루프)
  • 결과 페이지 하단: "전문가 상담으로 진짜 인연 찾기" CTA

듀오 사례 벤치마킹

참고 전략

듀오의 "내 상황에 가입비가 얼마?" 테스트가 바이럴에 성공한 구조를 참고합니다. 간단한 정보 입력 후 맞춤 결과를 제공하여 상담 전환으로 연결하는 퍼널입니다.

핵심 구조: 가벼운 테스트 진입 → 흥미로운 결과 → SNS 공유 → 결과 페이지에서 상담 신청 CTA. 테스트 참여자 데이터는 잠재 고객 DB로 활용 가능.
