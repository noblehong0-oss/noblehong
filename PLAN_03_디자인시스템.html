시그니처 그라데이션

Gold → Purple — 브랜드 시그니처. 히어로 오버레이, 구분선 악센트, 호버 보더에만 제한 사용.
Depth Gradient — 섹션 간 배경 깊이감. Warm White ↔ Warm Linen 교차로 레이어 구분.
Gold Glow — CTA/통계 섹션 배경. 라이트 배경 위 골드 빛이 은은하게 퍼지는 효과. opacity 낮게 유지.

CSS 변수 정의

:root {
  /* Background (Light) */
  --bg: #faf9f7;
  --surface: #ffffff;
  --surface-elevated: #f3f1ed;
  --border: #e5e0d8;
  --border-hover: #d4cec4;

  /* Text (Dark on Light) */
  --text-primary: #1a1714;
  --text-secondary: #5c564e;
  --text-muted: #9e9893;

  /* Accent */
  --gold: #a08030;
  --gold-light: #c4a55a;
  --gold-dark: #7a6020;
  --purple: #7b2d8e;
  --purple-light: #9b4dae;

  /* Gradient */
  --gradient-signature: linear-gradient(135deg, #c4a55a, #7b2d8e);
  --gradient-glow: radial-gradient(ellipse at center, rgba(160,128,48,0.08), transparent 70%);
}

Section 05

모션 토큰

모든 트랜지션과 애니메이션에 적용할 타이밍 규격.

토큰 Duration Easing 용도
instant 100ms ease-out 호버 컬러 전환, 포커스 보더
fast 200ms ease-out 버튼 호버, 아이콘 전환
normal 400ms cubic-bezier(0.16, 1, 0.3, 1) 카드 진입, 메뉴 열림
slow 800ms cubic-bezier(0.16, 1, 0.3, 1) 섹션 리빌, 커튼 전환
dramatic 1200ms cubic-bezier(0.22, 1, 0.36, 1) 히어로 진입, Lottie 스크롤 동기화
스태거 규칙: 같은 레벨 요소가 순차 진입할 때 0.1~0.15s 간격. 최대 5개까지만 스태거. 6개 이상은 2줄로 나눠 행 단위로 스태거.
