NS Migration · 2026-05-12

noblehong.com 네임서버 이전 가이드

카페24 NS → Cloudflare NS 전환. 카페24 호스팅·FTP·CRM은 그대로 살아있고, DNS 관리권만 Cloudflare로 옮깁니다.

✅ 자동으로 끝난 작업

  • ✓Cloudflare Zone 생성 (noblehong.com)
  • ✓CF DNS 레코드 4개: apex / www / admin / crm
  • ✓Vercel 도메인 등록 3개: noblehong.com, www, admin
  • ✓코드 내 카페24 URL → crm.noblehong.com 치환 (about.html, _worker)
  • ✓Worker secret CRM_ENDPOINT 새 URL로 업데이트
  • ✓.env / wrangler ALLOWED_ORIGINS 갱신

STEP 1. Cloudflare NS 값 복사

아래 2개 값을 카페24 NS 변경 화면 1차 / 2차 슬롯에 입력합니다.

1차 네임서버

hope.ns.cloudflare.com

2차 네임서버

sonny.ns.cloudflare.com

STEP 2. 카페24 NS 변경 화면 입력

카페24가 4개 슬롯을 요구하면 같은 값을 반복 입력해도 OK. IP 칸은 비워두세요.

슬롯 호스트명 IP
1차 hope.ns.cloudflare.com 비움
2차 sonny.ns.cloudflare.com 비움
3차 비움 또는 hope 반복 비움
4차 비움 또는 sonny 반복 비움
  • "Whois 잠금" / "도메인 잠금" 옵션이 켜져 있으면 일시 해제 후 변경
  • 카페24 변경 후 → 가비아(실제 registrar)로 위임이 자동 전달됨
  • 카페24에서 변경이 안 되면 가비아(gabia.com) 로그인 후 동일 절차

STEP 3. 전파 확인 (1~6시간)

PowerShell에서 다음 명령으로 NS 전파 확인: 

Resolve-DnsName noblehong.com -Type NS -Server 8.8.8.8

→ hope.ns.cloudflare.com, sonny.ns.cloudflare.com이 보이면 전파 완료.

또는 웹에서 확인:

whatsmydns.net에서 NS 전파 보기 →

STEP 4. 새 사이트 작동 검증

  • https://noblehong.com → 새 Vercel 사이트 정상 노출
  • https://www.noblehong.com → apex로 라우팅 (또는 동일 사이트)
  • https://admin.noblehong.com → 관리자 페이지 노출
  • https://crm.noblehong.com/include/inc/form/proc_consult.html → 카페24 PHP 정상 응답
  • 홈페이지 간편문의 제출 → 텔레그램 알림 수신 + 카페24 CRM에 데이터 도달
